# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅                |
| 0.2.x   | ✅                |
| 0.1.x   | ❌                |

## Reporting a Vulnerability

Fix security vulnerabilities once a week, and do not update security fixes if they are not found

Report the Vulnerabilities in Issues page
